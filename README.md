# pengumpulan_Tugas12_Rakan_Gustian_Muhammad_Akbar_301250020
Sistem Nilai Mahasiswa (Python)

Aplikasi Command Line Interface (CLI) berbasis Python untuk mengelola data nilai mahasiswa dengan penyimpanan menggunakan file CSV.
 Fitur:
Menambahkan data nilai mahasiswa

Menghitung nilai rata-rata secara otomatis

Menampilkan seluruh data mahasiswa dengan format yang jelas

Mencari data mahasiswa berdasarkan nama

Penyimpanan data permanen menggunakan file CSV
Struktur Proyek
Sistem-Nilai-Mahasiswa/
│
├── main.py        # Program utama
└── nilai.csv      # File penyimpanan data nilai

Format Data CSV

Data disimpan dalam file nilai.csv dengan format berikut:

Nama,Tugas,UTS,UAS,RataRata
rakan gustian,88.0,90.0,89.0,89.00
reval,87.0,86.0,85.0,86.00

Cara Menjalankan Program

Pastikan Python sudah terinstal di komputer

Clone repository ini atau unduh file proyek

Jalankan perintah berikut melalui terminal atau command prompt:

python main.py

 Menu Program
Saat program dijalankan, akan muncul menu berikut:

=== SISTEM NILAI MAHASISWA ===
1. Tambah Data
2. Tampilkan Semua Data
3. Cari Data Mahasiswa
4. Keluar

Contoh Output Pencarian Data
=== DATA MAHASISWA ===
Nama        : rakan gustian
Nilai Tugas : 88.0
Nilai UTS   : 90.0
Nilai UAS   : 89.0
Rata-rata   : 89.00
Teknologi yang Digunakan

Bahasa Pemrograman: Python

Penyimpanan Data: File CSV
